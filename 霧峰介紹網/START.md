# 快速啟動指南

## 方式一：直接開啟（最簡單）

1. 在檔案總管中找到 `index.html`
2. 雙擊打開，網站會在預設瀏覽器中開啟
3. 開始瀏覽霧峰旅遊網站！

## 方式二：使用本地伺服器（推薦）

### 使用 Python（如果已安裝）

```bash
# 在專案資料夾中開啟命令提示字元 (cmd) 或 PowerShell
# 然後執行：
python -m http.server 8000
```

然後在瀏覽器開啟: `http://localhost:8000`

### 使用 Live Server（VS Code 擴充功能）

1. 在 VS Code 中開啟專案資料夾
2. 安裝 "Live Server" 擴充功能
3. 右鍵點擊 `index.html`，選擇 "Open with Live Server"
4. 網站會自動在瀏覽器中開啟

## 常見問題

### Q: 圖片或影片無法顯示？
A: 確保 `Website Materials` 資料夾與 HTML 檔案在同一層級。

### Q: 輪播圖不會自動切換？
A: 請確認 JavaScript 檔案已正確載入，開啟瀏覽器開發者工具 (F12) 檢查是否有錯誤訊息。

### Q: 手機版選單無法展開？
A: 請確認使用的是現代瀏覽器（Chrome, Firefox, Safari 最新版）。

### Q: 影片無法播放？
A: 某些瀏覽器可能需要本地伺服器才能播放影片，請使用「方式二」啟動網站。

## 開發者模式

如果想要修改網站，可以：

1. **修改文字內容**: 直接編輯 HTML 檔案
2. **修改樣式**: 編輯 `css/style.css`
3. **修改互動功能**: 編輯 `js/main.js`
4. **替換圖片**: 將新圖片放入 `Website Materials/Image/`，並確保檔名一致
